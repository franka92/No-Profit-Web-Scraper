Il progetto OnDa

OnDa è un progetto sviluppato nell'ambito di un lavoro di tesi, in collaborazione con l'Università di Bologna.
Lo scopo del progetto consisteva nel raccogliere le informazioni relative alle Associazioni Non Profit al fine di creare un dataset ricco ed omogeneo, strutturato secondo un modello standard che ne permettesse il libero utilizzo.
Il dataset creato e pubblicato attraverso questa pagina può essere utilizzato come base su cui modellare dei servizi o semplicemente come punto di consultazione.

Come accedere ai dati

SPARQL End Point

Si tratta di un Web Service messo a disposizione per interrogare il dataset utilizzando il linguaggio SPARQL.
Lo SPARQL Endpoint è raggiungibile al seguente indirizzo: http://localhost:3030

Web App

Tramite la pagina web raggiungibile al link http://localhost/TESI/views/onda.html è possibile consultare i dati in maniera semplice e visualizzare utili grafici.
